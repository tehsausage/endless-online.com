Each theme is a package of 3 indentical named files that should be placed inside the gfx/themes folder of your game client. For example:

./gfx/themes/mynewtheme000.ini - text file to enter the name, autor, customized font colors ..etc
./gfx/themes/mynewtheme001.egf - duplicated gfx001.egf file to modify with customized graphics
./gfx/themes/mynewtheme002.egf - duplicated gfx002.egf file to modify with customized graphics
