Endless Online : Getting Started
How to still play Endless Online on a cloned or active third-party server? Here is how:
1. Client Download
EndlessOnline is a standalone program, download and install the program first.

Or find the software package on the website of the third-party game server.

You can find third-party servers here
2. Configuration
Please run econfig.exe, or edit the configuration text file in the config directory.

Usually this is already okay when downloading a full modified client.

3. Find Old Friends
Find out who of your old friends are still around on Discord

Click here to visit the Endless Online discord community..
