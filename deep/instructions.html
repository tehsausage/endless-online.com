Instructions: Manual Deep Patch Installation
How to update your client to 0.3.X
Updates are offered as a pack of optional executable replacements. This way, players and server admins are free to try.
Do not forget to backup your current Endless client files before making changes.

NON EO-CLONE PLAYERS
Please wait and use the regular update program of your server.

1. Download and Unzip


Download the latest client and unzip it in your local Endless Online folder.
This will overwrite some of the current files.



2. Check /sfx folder


Make sure the following new folders have been added inside the sfx folder.



3. Check /gfx folder


Make sure the following new folders have been added inside the sfx folder.



4. Run EgfUpdater.exe


One of the files you unzipped is EgfUpdater.exe
Locate the file and double-click it to run the required update.
The program will ask for permission because it will make file-changes.



5. Running EgfUpdater.exe


The EgfUpdater.exe should tell you all is ready to update.
Press on 'Update Now' button to continue (takes a minute to complete)



6. Complete


The Updater tool should notify you that the upate was completed.



6. Ready


Now is a great time to run EConfig.exe to setup the new features the way you like.
Your updated client is ready.

( optional: feel free to delete EgfUpdater.exe and /gfx-update folder )
